# Audio Description Evaluator

A gamified learning platform for evaluating audio description quality.

## Prerequisites

- [Node.js](https://nodejs.org/) (version 14 or higher)
- npm (comes with Node.js)

## Setup & Run

1. **Clone the repository**
   ```bash
   git clone https://github.com/YOUR_USERNAME/AD-evaluator-gamify.git
   cd AD-evaluator-gamify
   ```
2. **Set up firebase**
   ```bash
   cd AD-evaluator-gamify/firebase/config.js
   ```
   - Set up config.js file

3. **Install dependencies**
   ```bash
   npm install
   ```

4. **Start the app**
   ```bash
   npm start
   ```

5. **Open your browser**
   - Go to `http://localhost:3000`
   - Sign up with a username and password
   - Start evaluating videos!
